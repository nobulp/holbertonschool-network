# Holberton School - Networking Basics

## Description

This project covers the fundamentals of computer networking, including the OSI model, types of networks, IP addressing, and transport protocols.

## Learning Objectives

- OSI Model: what it is, its 7 layers, and how it is organized
- LAN: typical usage and geographical size
- WAN: typical usage and geographical size
- Internet: what it is and how it works
- IP addresses: the 2 types (public and private)
- Localhost and subnets
- Why IPv6 was created
- TCP vs UDP: differences and use cases
- Ports: SSH (22), HTTP (80), HTTPS (443)
- ICMP/ping: checking network connectivity

## OSI Model Layers

| Layer | Name         | Description                          |
|-------|--------------|--------------------------------------|
| 7     | Application  | HTTP, SMTP, SNMP, FTP                |
| 6     | Presentation | Encryption, compression, formatting  |
| 5     | Session      | Session management                   |
| 4     | Transport    | TCP, UDP                             |
| 3     | Network      | IP, ICMP, routing                    |
| 2     | Data Link    | MAC addresses, Ethernet              |
| 1     | Physical     | Electrical signals, cables, radio    |
